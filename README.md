# 🏆 Sistema de Pronósticos CAI

![Dashboard del Sistema](ruta_a_la_imagen_que_subiste.jpg)

Un sistema integral de escritorio desarrollado para la gestión de pronósticos deportivos, estadísticas avanzadas de usuarios y seguimiento de resultados. El aplicativo permite llevar un control exacto de puntajes, rachas históricas, perfiles de decisión (optimismo/pesimismo) y la evolución gráfica de las posiciones a lo largo de los diferentes torneos.
